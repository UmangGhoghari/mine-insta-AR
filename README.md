# Assets Directory

This folder contains the image files for your AR menu app.

## Required Files:

1. **menu-image.jpg** - The digital menu image that will appear in AR
   - Replace this with your actual menu image
   - Recommended size: 1000x1400 pixels
   - Format: JPG or PNG
   - Aspect ratio: 1:1.4 (width:height)

2. **loading.gif** (Optional) - Custom loading animation
   - If you want a custom loading spinner
   - Not required - the app uses CSS animation by default

## How to Add Files:

1. Prepare your menu image (JPG format recommended)
2. Name it exactly `menu-image.jpg`
3. Place it in this assets folder
4. Do not change the file path in index.html

## Note:

The actual image files will be added manually by you. This README is just a placeholder.
